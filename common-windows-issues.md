# Common Windows Issues & Fixes

## 1. Slow Computer

### Cause:
Too many background processes or temp files

### Solution:
- Clear temp files
- Disable startup programs
- Restart system


## 2. No Internet Connection

### Cause:
Network issue or wrong IP

### Solution:
- Restart router
- Run:
  ipconfig /release
  ipconfig /renew
- Check network cables


## 3. Printer Not Working

### Cause:
Spooler service stopped

### Solution:
- Restart Print Spooler service
- Reinstall printer drivers


## 4. Application Not Opening

### Cause:
Corrupted files

### Solution:
- Reinstall application
- Run as administrator


## 5. System Running Slow

### Solution:
- Open Task Manager
- End unnecessary tasks
- Scan for viruses
